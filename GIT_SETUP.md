# Hướng dẫn cài đặt Git và Push code lên GitHub

## Bước 1: Cài đặt Git

1. Truy cập: https://git-scm.com/download/win
2. Tải file cài đặt (Git for Windows)
3. Chạy file cài đặt và làm theo hướng dẫn (giữ nguyên các tùy chọn mặc định)
4. Sau khi cài xong, mở lại terminal/PowerShell

## Bước 2: Tạo GitHub Repository

1. Truy cập: https://github.com
2. Đăng nhập (hoặc tạo tài khoản mới)
3. Click nút "+" ở góc trên bên phải → "New repository"
4. Đặt tên repository (ví dụ: `nightcord-app`)
5. Chọn **Public** (để người khác có thể xem)
6. **KHÔNG** tích "Add a README file" (vì bạn đã có code rồi)
7. Click "Create repository"

## Bước 3: Copy Repository URL

Sau khi tạo xong, GitHub sẽ hiển thị URL dạng:
```
https://github.com/your-username/nightcord-app.git
```
Copy URL này lại.

## Bước 4: Chạy các lệnh Git

Sau khi cài Git và tạo repo, quay lại đây và tôi sẽ giúp bạn push code!

---

## Hoặc: Sử dụng GitHub Desktop (Dễ hơn, không cần terminal)

1. Tải GitHub Desktop: https://desktop.github.com
2. Đăng nhập bằng GitHub account
3. File → Add Local Repository → Chọn folder `nightcord-app`
4. Commit changes
5. Publish repository lên GitHub
